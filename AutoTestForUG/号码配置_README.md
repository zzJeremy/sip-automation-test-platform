# 号码配置.xlsx 使用说明

## 文件概述
此Excel文件包含700个SIP账号配置，账号范围从670100到670799，用于SIP设备测试。

## 列说明
- **FXS账号**: 用于FXS模拟用户的账号（交替填充）
- **IP账号**: 用于IP用户的账号（交替填充）
- **手机号码**: 号码字段，与账号相同
- **密码**: 符合安全规则的密码
- **密码类型**: 密码的字符类型分类

## 密码规则
- 长度：4-12个字符
- 字符集：数字(0-9)、英文字母(a-z, A-Z)、英文标点符号
- 覆盖9种字符类型组合：
  1. 纯数字类型
  2. 纯小写字母类型
  3. 纯大写字母类型
  4. 纯大小写混合字母类型
  5. 纯英文标点符号类型
  6. 数字+字母组合类型
  7. 数字+标点组合类型
  8. 字母+标点组合类型
  9. 数字+字母+标点三者混合组合类型

## 使用方法
1. 直接将此Excel文件导入到被测设备
2. FXS账号列的数据用于FXS用户配置
3. IP账号列的数据用于IP用户配置
4. 手机号码列包含对应的号码信息
5. 密码列包含对应的密码信息

## 账号分布
- 总计700个账号
- 从670100到670799
- FXS账号和IP账号交替分配，确保两种类型的账号都有覆盖
