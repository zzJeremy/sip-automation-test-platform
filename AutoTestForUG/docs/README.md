# AutoTestForUG - SIP软交换系统自动化测试平台

## 项目概述

AutoTestForUG是一个专门用于SIP软交换系统自动化测试的平台，旨在提供全面的SIP协议测试、性能监控和业务流程验证功能。

## 项目结构

```
AutoTestForUG/
├── test_client/          # SIP测试客户端
├── test_server/          # SIP测试服务器
├── monitor_client/       # 性能监控客户端
├── business_test/        # 业务测试套件
├── config/              # 配置文件
├── utils/               # 工具函数
├── docs/                # 文档
└── main.py              # 主程序入口
```

## 功能特性

- **SIP协议测试**：支持SIP注册、呼叫建立、消息发送等测试
- **性能监控**：实时监控CPU、内存、网络等系统资源
- **业务流程测试**：完整的业务场景测试套件
- **配置驱动**：通过配置文件灵活管理测试参数
- **并发测试**：支持多用户并发测试场景
- **详细报告**：生成全面的测试报告和统计信息

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 配置测试参数：修改config/config.ini文件
2. 定义测试用例：在config/test_cases.ini中添加测试用例
3. 运行测试：执行main.py启动测试流程

## 配置说明

### config.ini
主要配置文件，包含系统参数、SIP服务器信息、监控阈值等。

### test_cases.ini
测试用例定义文件，包含各种SIP测试场景的参数。

## 测试流程

1. 系统初始化
2. 加载配置和测试用例
3. 启动性能监控
4. 执行SIP测试
5. 生成测试报告

## 模块说明

### test_client
SIP客户端模块，负责发起SIP请求、处理响应、模拟用户行为。

### test_server
SIP服务器模块，模拟SIP服务器行为，处理客户端请求。

### monitor_client
系统性能监控模块，监控CPU、内存、网络等资源使用情况。

### business_test
业务测试模块，执行复杂的业务流程测试。

### utils
通用工具函数模块，提供日志、文件处理、数据格式化等功能。

## 扩展功能

- 支持多种SIP消息类型测试
- 可配置的性能阈值告警
- 详细的日志记录和错误处理
- 多格式测试报告输出

## 维护说明

项目遵循模块化设计，便于功能扩展和维护。各模块职责明确，通过配置文件进行参数管理，降低了模块间的耦合度。
