# SIP自动化测试配置文件生成系统 - 详细设计文档

## 1. 项目概述

### 1.1 项目背景
本项目旨在为SIP自动化测试提供灵活、可配置的账号密码管理系统，支持多种测试场景，包括正常登录和异常处理验证。

### 1.2 项目目标
- 自动生成符合规则的SIP账号密码配置文件
- 支持XLSX格式的配置文件，便于人工编辑和设备导入
- 生成SIPP格式的认证文件，用于自动化注册测试
- 支持多IP账号场景，适应复杂网络环境

## 2. 系统架构

### 2.1 整体架构
```
[规则定义] -> [XLSX配置生成] -> [SIPP CSV生成] -> [测试执行]
```

### 2.2 核心组件
1. **XLSX配置文件生成器** - 生成设备配置文件
2. **SIPP认证文件生成器** - 生成SIPP测试文件
3. **多账号支持模块** - 处理单号码多IP账号场景

## 3. 功能规格说明

### 3.1 XLSX配置文件生成规则

#### 3.1.1 账户类型配置
- **默认设置**：全部为IP账户
- **FXS账户**：FXS帐号列全部留空
- **IP账户**：所有账号填入ip帐号列

#### 3.1.2 字段配置规则
| 字段 | 配置规则 | 示例 |
|------|----------|------|
| 手机号码 | 与账号数值相同 | 670100 |
| FXS端口 | 保持空值 | "" |
| 默认路由计划 | 固定值"高级路由计划" | 高级路由计划 |
| 高级路由计划 | 固定值"全呼计划" | 全呼计划 |
| 低级路由计划 | 固定值"全呼计划" | 全呼计划 |
| 绑定IP账号数量 | 根据IP账号列自动计算 | 1,2,3... |
| 来显方式 | 随机FSK或DTMF | FSK/DTMF |
| 反极控制 | 固定值"禁用" | 禁用 |
| 业务密码 | 固定值"1234" | 1234 |
| 注册密码 | 正常账号为账号后4位，异常账号为非法密码 | 0100, ab!@... |
| 彩铃 | 保持空值 | "" |

#### 3.1.3 账号范围
- **正常账号**：670100-670799（共700个）
- **异常账号**：780001-780100（共100个，用于验证非法密码处理）

### 3.2 异常密码生成规则

#### 3.2.1 密码类型
1. **长度异常**
   - 太短：1-3位密码
   - 太长：13-20位密码
2. **字符异常**
   - 包含非法字符：!@#$%^&*()+=[]{}|\\:;'"<>?,./~`
   - 纯字母无数字：仅包含a-zA-Z
3. **混合异常**
   - 长度过长且包含非法字符

### 3.3 SIPP认证文件生成规则

#### 3.3.1 基本格式
```
SEQUENTIAL
手机号;[authentication username=用户名 password=密码]
```

#### 3.3.2 多IP账号支持
- 当IP账号列包含多个账号时（逗号分隔）
- 为每个IP账号生成独立的认证记录
- 所有记录共享相同的注册密码

#### 3.3.3 示例
```
SEQUENTIAL
670100;[authentication username=670100 password=1234]
670101;[authentication username=670101 password=5678]
670101;[authentication username=670102 password=5678]  # 多IP账号场景
```

### 4.3 数据流处理

#### 4.3.1 XLSX生成流程
1. 初始化Excel工作簿
2. 设置表头和样式
3. 生成正常账号数据（670100-670799）
4. 生成异常账号数据（780001-780100）
5. 应用所有配置规则
6. 保存文件

#### 4.3.2 SIPP文件生成流程
1. 读取XLSX配置文件
2. 解析IP账号列内容
3. 为每个IP账号生成认证记录
4. 输出SIPP格式文件

## 10. 全覆盖密码组合功能说明

### 10.1 功能概述
v1.4版本增加了全覆盖密码组合生成功能，可以生成覆盖所有支持的长度和输入种类组合的密码，用于更全面的测试验证。

### 10.2 正常账号密码全覆盖规则
- **长度覆盖**: 4-12位所有长度
- **字符类型覆盖**:
  - 纯数字
  - 纯小写字母
  - 纯大写字母
  - 纯标点符号
  - 混合大小写字母
  - 数字+字母（小写）
  - 数字+字母（大写）
  - 数字+字母（混合大小写）
  - 数字+标点符号
  - 字母+标点符号（混合大小写）
  - 数字+字母+标点符号（完整组合）
- **生成数量**: 每种长度11种字符组合，共99种有效密码组合

### 10.3 异常账号密码全覆盖规则
- **长度异常**:
  - 长度不足：1-3位密码
  - 长度超限：13-20位密码
- **字符异常**:
  - 包含特殊字符
  - 纯字母无数字
  - 空密码
  - 纯空格密码
- **生成数量**: 36种无效密码组合

### 10.4 配置文件生成
- **正常账号范围**: 670100-670198（共99个）
- **异常账号范围**: 780000-780035（共36个）
- **总计**: 135条记录
- **格式**: XLSX文件和对应的SIPP CSV文件
